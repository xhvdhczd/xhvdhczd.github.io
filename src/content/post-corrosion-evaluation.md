# 基于数据驱动的管道腐蚀厚度反演

把脉冲涡流信号**量化**成壁厚，是无损检测落地的关键一步。

## 问题定义

给定一组提取的特征向量 `x`，预测剩余壁厚 `d`：

```math
d = f(x) + ε
```

其中 `ε` 为模型与测量误差。

## 特征选择

常用特征包括：

1. 首个峰值时刻 `t_peak`
2. 特征时间常数 `τ`
3. 峰值幅值比
4. 频谱质心

## 模型对比（仿真标定集）

| 模型 | 平均误差 (mm) | 训练耗时 |
| --- | --- | --- |
| 线性回归 | 0.42 | 极短 |
| 随机森林 | 0.21 | 中等 |
| 多层感知机 | 0.18 | 较长 |

## 误差来源

- 提离（lift-off）变化
- 材料电导率波动
- 标定样本不足

## 结论

数据驱动方法能显著提升反演精度，但**可解释性**与**鲁棒性**仍需结合物理模型共同约束。
